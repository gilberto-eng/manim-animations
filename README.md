# 🎬 Manim Animations

Coleção de animações matemáticas e científicas criadas com **Manim**, focadas em visualização intuitiva, explicações claras e impacto visual.

---

## 🚀 Sobre o projeto

Este repositório reúne projetos de animação desenvolvidos com o **Manim**, explorando conceitos de matemática, física e ciência de forma visual e envolvente.

A ideia é transformar conceitos abstratos em algo **visualmente compreensível e interessante**.


---

## ⚙️ Como rodar

### 1. Instalar dependências

```bash
pip install manim
```

### 2. Executar uma animação

```bash
manim -pqh script.py NomeDaCena
```
